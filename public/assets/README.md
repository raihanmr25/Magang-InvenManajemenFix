# DOWNLOAD LOGO

Untuk menggunakan fitur PDF Sticker, Anda perlu mengunduh logo Lambang Kota Semarang.

## Link Download Logo

**Wikimedia Commons (Recommended):**
https://upload.wikimedia.org/wikipedia/commons/thumb/9/9d/Coat_of_arms_of_Semarang%2C_Central_Java.svg/1200px-Coat_of_arms_of_Semarang%2C_Central_Java.svg.png

**Atau cari di Google:**
"Lambang Kota Semarang PNG"

## Instruksi

1. Download logo dari link di atas
2. Simpan dengan nama: `Lambang_Kota_Semarang.png`
3. Letakkan file di folder ini: `public/assets/Lambang_Kota_Semarang.png`

## Spesifikasi Logo

- Format: PNG (dengan background transparan lebih baik)
- Ukuran: Minimal 200x200 pixel
- Nama file: `Lambang_Kota_Semarang.png` (case-sensitive)

## Verifikasi

Setelah meletakkan logo, pastikan path berikut dapat diakses:
```
http://localhost:8000/assets/Lambang_Kota_Semarang.png
```

Jika logo sudah ada, fitur "Cetak Stiker" akan berfungsi dengan baik.
